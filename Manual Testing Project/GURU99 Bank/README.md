Guru99 Bank Web Application Testing Project

Overview

This project demonstrates manual functional testing of the Guru99 Bank demo web application.
It focuses on validating core banking workflows, including authentication, account management, and transaction processes.

The testing approach ensures that critical banking features work as expected from an end-user perspective while maintaining data integrity and proper system behavior.

Base Information

Application: Guru99 Bank Demo Application
URL: http://demo.guru99.com/V4/
Testing Type: Manual Functional Testing
Domain: Banking Web Application
Scope: Core Banking Modules

Project Objectives

- Validate core banking functionalities
- Test user authentication and login security
- Verify account creation and management features
- Validate financial transactions (deposit and withdrawal)
- Ensure correct system responses for valid and invalid inputs
- Identify defects and ensure application stability
- Simulate real-world user banking scenarios

Modules Tested

* Login Module – User authentication validation
* New Customer Module – Customer creation functionality
* Edit Customer Module – Updating customer details
* Delete Customer Module – Customer removal
* New Account Module – Account creation
* Deposit Module – Adding funds to account
* Withdrawal Module – Deducting funds from account
* Fund Transfer Module – Transferring money between accounts
* Balance Enquiry Module – Checking account balance

Test Approach

* Manual functional testing of each module
* Positive and negative test case validation
* UI and form validation testing
* Boundary and input validation testing
* End-to-end workflow validation across modules
* Verification of system messages and alerts

Test Scenarios Covered

- Login with valid credentials
- Login with invalid credentials
- Create new customer with valid data
- Edit existing customer details
- Create new account for customer
- Deposit money into account
- Withdraw money from account
- Transfer funds between accounts
- Check account balance accuracy
- Delete customer and validate removal

Test Data

Valid and invalid login credentials

Account types (Savings, Current)
Transaction amounts for deposit and withdrawal
